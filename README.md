## 🌌 About the Project

This is my personal portfolio reimagined as a modern, premium **Bento Grid Dashboard**. Built with pure front-end technologies, it features fluid animations, glassmorphism, and responsive states designed to provide a smooth user experience across desktop and mobile devices.

### ✨ Features
- **Premium Glassmorphism**: Semi-transparent boxes with background blur and subtle hover glow animations.
- **Dynamic Glow Background**: CSS-driven orbital animations creating a premium fluid atmosphere.
- **Project Catalog**: Highlighted projects such as *PixInspect*, *AirCanvas*, *Voyager*, *VisionGlobe*, and *AirMouse*.
- **Interactive Experience & Education Timeline**: A modular layout displaying professional and community leadership experience.
- **Live GitHub Metrics**: Real-time integration of GitHub contribution statistics and stats cards.
- **Responsive Layout**: Fluidly converts into a double-column grid or single-column layout on mobile screens.

## 🛠️ Built With

- **HTML5** & **Vanilla CSS3** (Custom keyframes, CSS Variables, Bento Grid flex systems)
- **JavaScript (ES6+)**
- **Google Fonts** (*Inter* & *Anton*)
- **FontAwesome Icons**
- **GitHub Readme Stats & Streak API**
